Designed and implemented a **wearable health monitoring system** using **ARM Cortex-M4**, integrating **pulse oximeter** and **temperature sensors** via **I2C** for continuous biometric data acquisition. Developed an **interrupt-driven alert mechanism** with **BLE transmission** to a mobile dashboard, ensuring real-time monitoring and reliable wireless communication.
